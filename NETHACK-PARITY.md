133 source files) to **Ascend**, the ASCII roguelike on Polkadot. Each NetHack
`src/*.c` file is roughly one subsystem; we inventory every one, mark Ascend's
status, and plan the missing parts in the Polkadot/web3 idiom.

## 1. Subsystem inventory (by NetHack source file)

### A. Character & attributes
- **`attrib.c`, `u_init.c`, `role.c`, `exper.c`** ‚Äî STR/DEX/CON/INT/WIS/CHA, role/race/gender/alignment, XP levels, attribute gain/loss, exercise. ‚Üí **‚ùå Ascend has no attributes, role, race, alignment, or XP.** Today the player is a single fixed avatar (20 HP, fixed damage). **Plan (Phase 6):** add the six attributes (Polkadot-flavored display: STR=Stake-weight, DEX=Latency, CON=Resilience, INT=Throughput, WIS=Insight, CHA=Reputation), an XP/level curve (levels = "epochs"), and per-archetype starting kits/stats. Attributes feed to-hit, AC, carry capacity, spell success, prayer.
- **`attrib.c` intrinsics** ‚Äî speed, see invisible, fire/cold/shock/poison/sleep/disint resistance, telepathy, teleport(itis/control), searching, warning, etc. ‚Üí **‚ùå none** (we have ad-hoc ring effects). **Plan (Phase 8):** an intrinsics bitset on the player, gained from corpses (eat ‚Üí intrinsic), rings, and XP; flavored as "protocol upgrades you've internalized."
- **`insight.c`** (enlightenment/conduct dump) ‚Üí **‚ùå**. **Plan:** an "audit report" (`#audit`) listing your attributes, intrinsics, vows, and on-chain stats.

## 4. Where Ascend already *exceeds* NetHack (keep leaning in)

- **On-chain economy** ‚Äî PAS is a real testnet asset; shops are real wallet transactions.
- **Tradeable NFT relics** ‚Äî `forge` mints artifacts as ERC-721s with on-chain rarity rolls; they persist across runs and trade on any marketplace. (NetHack artifacts are local save state.)
- **On-chain Hall of Fame + bones** ‚Äî your deaths seed *other players'* dungeons, permanently, trustlessly.
- **Live P2P co-op** ‚Äî two adventurers in one shared dungeon (NetHack is single-player), now **peer-authoritative deterministic lockstep** over a serverless WebRTC link: a shared seed + broadcast-on-consume keeps both in sync with no host "seat", each client rendering locally. **Independent floors** (each adventurer fully simulated on its own floor), per-adventurer logs, **signal chat** (distance/LOS-degraded, banner over the map), a **queued-action** badge with LIFO undo, and a race mode ‚Äî none of which has a NetHack analog. **Partners are friendly NPCs:** walking into each other **slips past** (swaps tiles) rather than auto-attacking, with the **Host holding the tile** in any same-tile dispute (a deterministic tie-break); you can still **turn on a partner deliberately** (Kick `K` into them) and **rays/AoE always strike anyone in the line of fire** ‚Äî so accidents and betrayals stay possible, but only by choice.
- **Procedural adaptive score** ‚Äî a zero-asset Web-Audio engine: an ambient bed + a rhythm groove that ebbs to ambience when calm, distortion that tracks the *visible threat* around you, and a melodic, BPM-locked combat-tension layer (trills + drums + bass) grown from each zone's own key. NetHack is silent.
- **Mobile** ‚Äî a touch control deck; NetHack is keyboard-bound.

Keep every new NetHack subsystem wired to this spine: if it touches value, make it on-chain; if it's an artifact, make it a tradeable relic; if it's a milestone, etch it in the Hall of Fame.
